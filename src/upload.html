Data Ingestion Engine

Refresh the warehouse with the latest Transactions, Households, and Products CSVs. File names must contain transaction, household, or product so the router can place them in the right table.

Drag & drop CSV files here

Accepted: 400_transactions.csv, 400_households.csv, 400_products.csv. You can drop multiple files at once.

How the loader works

  • Normalises column names the same way the initial Azure loader did (trim, replace whitespace with _).
  • Replaces the existing table contents atomically (disables FKs, truncates, bulk inserts in batches of 1,000).
  • Any upload here immediately updates the dashboard and the household search page.
